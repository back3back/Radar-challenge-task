## 雷达挑战任务3

### 3-1

仿真一个 1 发 4 收的阵列 ($d = \lambda/2$) 测量某点目标（自行设定目标的距离和角度）的角度。

**要求：**

1. 产生回波信号；
2. 给出用 FFT 测角和相位法测角的仿真结果；
3. 分析对比两种测角方式下的作用距离。
